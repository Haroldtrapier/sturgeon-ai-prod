## 🎬 RECORDING TIPS

**Tools Needed:**
- Screen recorder (Loom, OBS, or ScreenFlow)
- Microphone (for voiceover)
- Video editor (optional)

**Recording Steps:**
1. Open localhost:3000 in Chrome (clean window)
2. Set window size to 1920x1080
3. Close unnecessary tabs
4. Hide bookmarks bar
5. Use slow, deliberate movements
6. Pause 2 seconds between scenes
7. Highlight important buttons/features with cursor

**Editing Tips:**
- Add zoom effects on important features
- Add text overlays for key points
- Background music (low volume)
- Speed up installation/loading scenes (2x)
- Add chapter markers

**Export Settings:**
- Resolution: 1920x1080 (1080p)
- Frame rate: 30fps or 60fps
- Format: MP4 (H.264)
- Bitrate: 8-10 Mbps
